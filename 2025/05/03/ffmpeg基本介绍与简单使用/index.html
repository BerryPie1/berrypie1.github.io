FFmpeg基本介绍与简单使用

介绍

FFmpeg 是领先的多媒体框架，能够解码、编码、转码、多路复用、解多路复用、流式传输、过滤和播放人类和机器创造的几乎所有内容。它支持从最晦涩的古老格式到最前沿的格式。无论它们是由某个标准委员会、社区还是公司设计的。它还具有高度的可移植性： FFmpeg 可在 Linux、Mac OS X、Microsoft Windows、BSD、Solaris 等多种构建环境、机器架构和配置下编译、运行并通过我们的测试基础架构 FATE。

ffmpeg命令行工具

ffmpeg命令行工具官方文档：ffmpeg Documentation

ffmpeg是一个通用的媒体转换器，读取各种输入，过滤并转码为各种输出格式。

ffmpeg通过-i选项（option）指定任意数量的输入，并写任意数量的输出。命令行中任何不能被解释为选项的内容均被视为输出url。每个输入和输出都可以包含任意数量不同类型的基本流（视频/音频/字幕/数据），但允许的流数量和类型可能受到容器格式的限制。自动选择或使用-map选项可以选择那些输入流将进入哪些输出流。

要在选项中引用输入/输出，必须使用它们的索引（从0开始），例如第一个输入为0，第二个输入为1。同样地，输出/输出中的流也通过索引来引用，例如2:3表示第三个输入或输出的第四个流。

一般来说，选项作用于下一个指定文件。因此，顺序很重要，同一选项可以在命令行中出现多次。每次出现的选项都会作用于下一个输入或输出文件。但全局选项（如冗长程度verbosity level）除外，它们应首先指定。
